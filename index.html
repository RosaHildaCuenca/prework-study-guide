HTML

  • The head element contains information about the webapage.
  • The body element represents the visible contant shown to the user.
  • The h1-h6 elements represent the level of heading a given text block represents.
  • The ul,ol,li elements represent unordered lists, ordered lists, and list items.
  • The img element contains information about images that are displayed on the webpage.
  • The p element represents a paragraph or block of text.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indictes how much space we want around the current inside an element.
  • DRY can greatly reduce the overall size and complexity of a codebase.

  • Declarations conatin: the CSS property and the value of the property.
